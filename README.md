# Midalytic Landing Page

This repository contains the source code for the Midalytic landing page.

## Project Structure

- `midalytic-landing/`: The new React + Tailwind CSS application.
- `index.html`: The old static HTML version (deprecated).

## Getting Started with the React App

1. Navigate to the project directory:
   ```bash
   cd midalytic-landing
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Features

- **Modern Stack**: Built with React, Vite, and Tailwind CSS.
- **Responsive Design**: Fully responsive layout for all devices.
- **Animations**: Smooth transitions using Framer Motion.
- **Portfolio Section**: Showcase of recent projects with generated visuals.
- **Premium UI**: High-quality design with custom fonts and colors.
